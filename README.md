TIK-TASK: APLICACION MOVIL PARA GUARDAR APUNTES DE TEXTO, VOZ E IMAGENES
·····················································································
Descripcion: El presente proyecto, es una iniciativa pensada por estudiantes y para estudiantes para disminuir la perdida de informacion 
clave en el aula, mediante la creacion de una app que permita almacenar esta informacion sensible de manera facil e intuitiva.
